# ResumeAI — Intelligent Career Tools

## Overview

ResumeAI is a modern, AI-powered web application designed to help users analyze and improve their resumes. It provides smart feedback, resume scoring, and interactive tools to enhance job readiness with a clean, Apple-inspired UI.

---

## Features

* Resume Analyzer
  Upload or paste your resume and get instant feedback with scoring.

* AI Suggestions
  Get intelligent recommendations to improve content, structure, and keywords.

* Performance Metrics
  View detailed scores for readability, skills, impact, and ATS compatibility.

* AI Tools Section
  Includes utilities like content enhancement, bullet rewriting, and more.

* AI Chat Assistant
  Ask career-related questions and get instant guidance.

* Modern UI/UX
  Clean Apple-style interface with smooth animations and aesthetic layout.

## How to Use

1. Open the project in your browser
2. Upload or paste your resume
3. Select a job role (optional)
4. Click Analyze
5. View results and improve your resume
